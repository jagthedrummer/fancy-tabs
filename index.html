FancyTabs

FancyTabs is a JavaScript library built on top of Prototype and Scriptaculous that provides application developers an easy way to include dynamic tabbed panels in their applictaion.

FancyTabs was originally conceived to provide a some what 'Eclipse like' interface with a file browser and multiple editor tabs within the aPeel content management system.
